Overview of NewsAPI

With NewsAPI, you can search and retrieve real-time items from more than 150,000 online sources using a straightforward HTTP REST API. It can assist you in responding to queries such as:

  • Which top stories are currently featured on TechCrunch?
  • Which fresh reports on the upcoming iPhone surfaced today?
  • Have any bloggers recently reviewed or highlighted my business or product?

A variety of parameters, including keywords, the date of publication, the source domain name, and language, can be used to search for articles. The popularity of the source, relevance to the search terms, and publication date can all be used to sort the results.

Features of NewsAPI

  • Top Headlines: Get the latest breaking news headlines for countries, categories, and specific publishers.
  • Everything: Search every article published by over 150,000 sources in the last five years. This is ideal for news analysis and article discovery.
  • Sources: Retrieve information about the most notable sources available for obtaining top headlines, including their name, description, and category.

How to Get Started with NewsAPI

  1. Step 1: Register for an API Key

    To start using NewsAPI, you need to register for an API key. Follow these steps:

    • Visit the NewsAPI registration page.
    • Fill in your First Name and Last Name.
    • Enter a valid Email Address. Note: Disposable email addresses are blocked.
    • Create a Password.
    • Select whether you are an individual or working on behalf of a business.
    • Agree to the terms and conditions, then submit the form.
    • Check your email for a verification message, and click the link to verify your account.
    • Once verified, you will receive your unique API key. Keep this key secure, as it identifies your requests. 
    const apiKey = 'YOUR_API_KEY';  // Replace with your actual API key.

  2. Step 2: Connect to the API

    Use the Fetch API in JavaScript to connect to NewsAPI. Below is an example that fetches the top headlines: 

    
const url = `https://newsapi.org/v2/top-headlines?country=us&apiKey=${apiKey}`;

fetch(url)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

  3. Step 3: Output Data on the Web Page

    Once you receive the data, you can display it on your webpage. Here’s how to loop through the articles and display them with links: 

    
fetch(url)
  .then(response => response.json())
  .then(data => {
    const articles = data.articles;
    articles.forEach(article => {
      const articleElement = document.createElement('div');
      articleElement.classList.add('news-article');

      const titleElement = document.createElement('h4');
      const linkElement = document.createElement('a');
      linkElement.href = article.url;
      linkElement.target = '_blank';  // Open link in a new tab
      linkElement.textContent = article.title;
      titleElement.appendChild(linkElement);
      articleElement.appendChild(titleElement);

      const descriptionElement = document.createElement('p');
      descriptionElement.textContent = article.description;
      articleElement.appendChild(descriptionElement);

      document.getElementById('news-container').appendChild(articleElement);
    });
  })
  .catch(error => console.error('Error:', error));

  4. Step 4: Customize the API Call

    NewsAPI provides different endpoints for various use cases. Here’s how you can use them:

    • Everything Endpoint: Search articles from over 150,000 sources in the last 5 years. Use it for news analysis and discovery. 
      • 
GET https://newsapi.org/v2/everything?q=Apple&from=2024-08-10&sortBy=popularity&apiKey=YOUR_API_KEY

// Example using curl
curl https://newsapi.org/v2/everything -G \
  -d q=Apple \
  -d from=2024-08-10 \
  -d sortBy=popularity \
  -d apiKey=YOUR_API_KEY
    • Top Headlines Endpoint: Get breaking news headlines for countries, categories, and publishers. Ideal for live news tickers. 
      • 
GET https://newsapi.org/v2/top-headlines?country=us&apiKey=YOUR_API_KEY

// Example using curl
curl https://newsapi.org/v2/top-headlines -G \
  -d country=us \
  -d apiKey=YOUR_API_KEY
    • Sources Endpoint: Retrieve information about notable sources for headlines, including name, description, and category.
